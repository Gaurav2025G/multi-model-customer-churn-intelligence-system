# 🚀 Multi-Model Customer Churn Intelligence System

> **An end-to-end Machine Learning solution for predicting customer churn using Logistic Regression, Random Forest, XGBoost, SHAP Explainable AI, and an interactive Streamlit dashboard.**

# 📌 Problem Statement

Customer churn is one of the biggest challenges for subscription-based businesses. Losing existing customers leads to reduced revenue and increased acquisition costs. Traditional approaches often fail to identify at-risk customers early, making proactive retention strategies difficult.

---

# ✅ Solution

Developed an end-to-end **Machine Learning-based Customer Churn Intelligence System** that predicts customer churn probability using customer demographics, service usage, contract information, and billing history. The solution combines predictive modeling, explainable AI, and interactive business dashboards to support data-driven customer retention decisions.

---

# ✨ Key Features

* Multi-Model Machine Learning Prediction
* Logistic Regression, Random Forest & XGBoost Comparison
* Customer Churn Probability Prediction
* SHAP Explainable AI
* Interactive Business Dashboard
* Executive Decision Support
* Downloadable Churn Reports
* Interactive Data Visualization

---

# 📊 Dataset

**Dataset:** Telco Customer Churn Dataset

* **7,043 Customer Records**
* **21 Features**
* Customer Demographics
* Service Information
* Contract Details
* Billing Information
* **Target Variable:** Customer Churn

---

# 🛠 Tech Stack

**Programming:** Python

**Machine Learning:** Scikit-learn, XGBoost

**Data Processing:** Pandas, NumPy

**Visualization:** Plotly, Matplotlib

**Explainable AI:** SHAP

**Deployment:** Streamlit

# 🤖 Machine Learning Models

* Logistic Regression
* Random Forest
* XGBoost

| Model               | ROC-AUC  | F1 Score |
| ------------------- | -------- | -------- |
| Logistic Regression | **0.86** | **0.64** |

# 🔍 Model Explainability

Implemented **SHAP (SHapley Additive Explanations)** to:

* Explain individual predictions
* Visualize feature importance
* Improve model transparency
* Support business decision-making

---

# 📈 Business Insights

The analysis identified several key factors influencing customer churn:

* Low Customer Tenure
* Month-to-Month Contracts
* High Monthly Charges
* Fiber Optic Internet Service
* Lack of Online Security & Tech Support
